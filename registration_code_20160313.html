Slab registration code for high resolution imaging

 

This software is dedicated to the registration of slabs for high resolution MR imaging of the hippocampus at 7T. It could also be used for the registration of slabs for any other application if the same kind of acquisitions are performed.
This code was developped by Linda Marrakchi-Kacem in the ARAMIS Lab (ICM, Brain and Spinal Cord Institute, CNRS, Inria, Inserm, Université Pierre et Marie Curie). For any use of this code, the following paper must be cited:

 

L Marrakchi-Kacem, A Vignaud, J Sein, J Germain, TR Henry, C Poupon, L Hertz-Pannier, S Lehericy, O Colliot, PF Van de Moortele, M Chupin. Robust imaging of hippocampal inner structure at 7T: in vivo acquisition protocol and methodological choices. MAGMA, 2016.

 

Before dowloading this code it is important to install an appropriate environment:

  • matlab which is needed to run SPM
  • SPM8 which can be downloaded from: http://www.fil.ion.ucl.ac.uk/spm/software/spm8/
  • BrainVISA which can be downloaded from: http://brainvisa.info/index_f.html

    Download

    The pack registration_pack.zip contains matlab and python codes and the bash scripts that should be launched in order to register the high resolution slabs. The file instructions_manual.pdf explains how to use the scripts.
